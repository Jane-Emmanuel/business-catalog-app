Business Catalog PWA
====================

This is a simple mobile-friendly product catalog Progressive Web App (PWA).

Features:
- Add, edit, and delete products (seller password = 1234 by default).
- Customers can view products and order via WhatsApp.
- Currency symbol switcher ($, ₦, €, £, ₹).
- Works offline and can be installed like an app.

How to Deploy:
1. Upload this folder to a GitHub repository.
2. Connect the repo to Netlify (New Site > Import from Git).
3. Netlify will auto-deploy. No build settings needed (static site).
4. Share the Netlify link with customers. They can "Add to Home Screen" on mobile.

Icons:
- icons/icon-192.png
- icons/icon-512.png

You can replace these icons with your own branding later.
